# RL - Atelier Reinforcement Learning

Ce repository contient le matériel du workshop « Reinforcement Learning ». Le contenu a été préparé pour IndabaX 2025.
Le contenu principal se trouve dans le dossier `workshop/`.

Principaux emplacements :
- `workshop/theorie/` : notes et chapitres théoriques.
- `workshop/pratiques/` : notebooks et séances pratiques.
- `workshop/_build/html/` : version HTML pré-générée du site (visualisable localement).

Pour démarrer rapidement (Linux / macOS) :

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r workshop/requirements.txt
```

Ensuite, ouvrez les notebooks dans `workshop/pratiques/` ou consultez le site HTML dans `workshop/_build/html/`.
